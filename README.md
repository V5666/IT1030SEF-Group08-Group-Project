# IT1030SEF-Group08-Group-Project
A lightweight, modern web application designed to help students recover lost items and report found belongings within the campus community.

Core Features
User Lifecycle: Full support for user registration, login, and secure logout.

Item Management: Users can post lost/found items with descriptions, images (Base64), and specific location tagging.

Search & Filtering: Dynamic filtering system that allows users to search items by keywords and specific districts (e.g., Hong Kong, Beijing).

Private Messaging: Integrated chat module allowing users to contact each other directly to coordinate item returns.

Admin Moderation: A dedicated administrative dashboard to review, approve, or reject pending posts to ensure content quality.
